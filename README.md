cvs_batch_data_extraction_tool

==============================

## ***软件功能***

本工具用于批量提取多个CSV文件中的指定字段（列），并自动汇总生成Excel或CSV结果文件。

适用于：

* 测试数据整理
* 设备数据统计
* 批量CSV数据分析
* 多文件数据汇总

支持：

√ 自动扫描文件夹中的CSV文件

√ 文件型号筛选（包含/排除）

√ 字段（列）筛选（包含/排除）

√ 批量提取指定字段

√ Excel输出

√ CSV输出

√ 汇总输出

√ 分文件夹输出

√ 自动生成处理进度

√ 自动跳过历史Result结果文件夹
