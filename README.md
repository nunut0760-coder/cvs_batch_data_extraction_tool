cvs_batch_data_extraction_tool

==============================
一、软件功能

本工具用于批量提取多个CSV文件中的指定字段（列），并自动汇总生成Excel或CSV结果文件。

适用于：

* 测试数据整理
* 设备数据统计
* 批量CSV数据分析
* 多文件数据汇总

支持：

√ 自动扫描文件夹中的CSV文件
√ 文件型号筛选（包含/排除）
√ 字段（列）筛选（包含/排除）
√ 批量提取指定字段
√ Excel输出
√ CSV输出
√ 汇总输出
√ 分文件夹输出
√ 自动生成处理进度
√ 自动跳过历史Result结果文件夹

==============================
二、使用步骤

step1：选择文件夹

点击【选择文件夹】按钮，选择存放CSV文件的大（主）文件夹。

程序会自动扫描该文件夹及其子文件夹中的CSV文件。

step2：筛选需要处理的文件（可选）

文件选择区域支持：

1）包含：
输入需要保留的型号关键词。

例如：

808

表示仅处理文件名中包含“808”的文件。

2）排除：
输入不需要处理的型号关键词。

例如：

808

表示忽略所有文件名中包含“808”的文件。

支持多个关键词：

808,909,1001（注意：分隔符为英文下的逗号）

step3：选择需要提取的字段

程序会自动读取CSV字段名称。

在“选择需要提取的字段(列)”区域：

勾选需要导出的字段。

支持：全选、取消全选

支持字段筛选：

1）包含：
筛选指定字段名称

2)排除：
排除指定字段名称

step4：选择处理模式

全部汇总：将所有CSV提取结果汇总到一个文件中。

分文件夹输出：按照子文件夹分别生成结果文件。

两种都输出：同时生成以上两种结果。

step5：选择输出格式

Excel (.xlsx)：生成Excel文件。

CSV (.csv)：生成CSV文件。

step6：开始处理

点击【开始处理】按钮。

程序将自动完成数据提取并显示处理进度。

==============================
三、结果文件说明

处理完成后，程序会自动生成结果目录。

例如：

Result1
Result2
Result3

每次运行都会自动创建新的结果目录，不会覆盖历史结果。

目录示例：

Result3

├─ Merged_Result.xlsx
├─ FolderA_Result.xlsx
├─ FolderB_Result.xlsx
└─ failed_files.txt

==============================
四、失败文件说明

如果某些文件读取失败：

* 编码异常
* 文件损坏
* 缺少指定字段

程序会自动记录到：

failed_files.txt

方便后续检查。

==============================
五、注意事项

1. 建议处理前关闭正在打开的CSV文件。

2. CSV文件列名尽量保持一致。

3. 程序会自动跳过：
   Result
   Result1
   Result2
   Result3
   等历史结果目录。

4. 输出文件默认保存到当前选择文件夹下的Result目录中。

5. 首次启动速度较慢属于正常现象。
